# 配置文件



#### 加载机制分三级

| 级别 | 位置         | 描述                                                         |
| ---- | ------------ | ------------------------------------------------------------ |
| 1    | 配置中心     | 所有服务公共配置，ZK，注册中心，网关，链路，架构统一维护     |
| 2    | 配置中心     | 各自系统配置，如功能开关，支持热刷新，支持多环境配置         |
| 3    | 本地开发环境 | bootstrap.properties，配置端口号，服务名，配置中心地址，基本不用改 |
